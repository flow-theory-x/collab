# パーリ経典ラノベ 登場キャラクター設定

> **共通シリーズ設定：** 法華曼荼羅と同一世界線のキャラクターだが、パーリ経典ラノベでは「神格化される前の人間としての姿」を描く。法華曼荼羅が「伝説になった後」なら、こちらは「伝説が生まれる前」の物語。

---

## ◆ シッダルタ（悉達多 / Siddhārtha Gotama）
別名：ブッダ（覚者）、タターガタ（如来）、後に弟子たちから「セゾン（世尊 / Bhagavān）」と呼ばれる
シャーキャ族の王子として生まれ、29歳で出家、35歳で悟りを開いた。
法華曼荼羅の「セゾン」が宇宙スケールのカリスマ俺様キャラだとすれば、こちらのシッダルタは**その原型**──穏やかだが目が笑っていない時がある、比喩の天才、問いかけで相手を追い詰めるソクラテス型の師。
超常的な奇跡ではなく、洞察と対話で問題を解く。怒らない、しかし逃げない。
弟子に「答え」を教えず「気づき」に導くのが流儀だが、弟子たちからは『ヒントが難解すぎて逆に迷う』と密かに不評。
80歳で人間として老い、病み、死ぬ。**その「死ぬ」という事実こそが、彼の最後の教え。**

> 📌 法華曼荼羅との接続：後世、弟子たちの記憶と信仰が積み重なり、「永遠の存在セゾン」として神格化されていく。パーリ版で描かれる人間シッダルタを知った上で法華版を読むと、セゾンの「方便」の意味が変わる。

---

## ◆ シッダルタの正式クルー（十大弟子）

### シャリア / シャーリプトラ（舎利弗 / Sāriputta）
ウィズダム（智慧第一）：シッダルタの右腕にして最も頭の切れる弟子。
元はサンジャヤ・ベーラッティプッタの弟子で、モクレンと共にシッダルタのもとに転向してきた。
冷静沈着で論理的だが、シッダルタの教えをロジカルに分析しすぎて**直感で悟った他の弟子に先を越される**ことがある。
シッダルタにはいつも軽くいじられているが、それが愛情からの照れ隠しであることを理解している。算数の成績はシッダルタより良い。
**パーリ版の特徴：** シッダルタより先に入滅する。その死は物語の重要な転換点──「智慧の権化」すら無常には勝てない。

### モクレン / モッガッラーナ（目連 / Moggallāna）
ソーサラー（神通第一）：超常的な能力を使いこなすチート野郎。
シャリアとは出家前からの親友で、二人セットでシッダルタの門を叩いた。
下手に神通力が使えるため、シッダルタの「ちょっと確かめてきてくれ」の一言で天界や地獄に飛ばされ、大抵ろくでもない目に遭う。
ただ、どんな無茶振りも神通力でなんとでもなるので、なんだかんだ楽しんでいる。
**パーリ版の特徴：** シャリアと同時期に亡くなる。しかもその死因は修行者としてではなく暴漢による暗殺──「神通力があっても、カルマからは逃れられない」という残酷な教訓を身をもって示す。

### カッサパ（摩訶迦葉 / Mahākassapa）
パンハンドラ（頭陀第一）：ストイックで規律を重んじる苦行の達人。
シッダルタが死んだ後、実質的リーダーとして第一結集（経典編纂会議）を主催する。
シッダルタの自由すぎる教え方に内心ヤキモキしているが、尊敬の念は誰より深い。
周囲にはストイックな聖人と思われているが、実は夜中にこっそり食べるスイーツがやめられなくて悩んでいる。
**パーリ版の特徴：** 第一結集でアーナンダを厳しく問い詰める役。「お前はまだ阿羅漢じゃないだろう」──この言葉がアーナンダの最後の覚醒を引き出す。

### アーナンダ（阿難 / Ānanda）
イーズドロッパー（多聞第一）：シッダルタの最側近にして、**本作のメイン語り部**。
記憶力は"人類最高レベル"。シッダルタの言葉を99.99%ロスレスで保存するが、肝心な時に限って動揺して忘れる。
イケメンで女性信者にモテすぎるのがちょっとした悩み（本人は全く気にしていないが、周囲は気にしている）。
**パーリ版の最重要設定：** シッダルタに最も近く、最も多くの教えを聞きながら、**師の生前には阿羅漢果に至れなかった唯一の主要弟子**。「わかるけどできない」──読者の感情移入先。
シッダルタの入滅後、第一結集の前夜にようやく悟る。30年間の「まだ足りない」が解けた瞬間は、本作最大のカタルシス。

### アヌルッダ（阿那律 / Anuruddha）
スヌーパー（天眼第一）：世界の真相を"見えてしまう"タイプの人。
瞑想中に居眠りしたのをシッダルタに叱られ、「二度と眠らない」と誓った結果、視力を失う。代わりに"天眼"を獲得するドラマチックな人生。
普段は控えめで優しいが、**ちょっと居眠りしたくらいで視力を失う展開には未だに納得していない。**
**パーリ版の特徴：** シッダルタの入滅の瞬間を天眼で正確に見届ける。「世尊は第四禅定から涅槃に入られた」──この証言が経典に刻まれる。

### カティ / カッチャーヤナ（迦旃延 / Kaccāyana）
トランスパイラ（論議第一）：難しい教えを"人間言語"に翻訳する伝道師。
シッダルタの抽象的な発言を「つまりこういうことですよね」とサマライズするのが得意。
カティが何か言うたびにシッダルタのこめかみの血管がピクピクするので周りはいつもヤキモキしているが、本人は全く気づいていない。
**パーリ版の特徴：** カッチャーヤナゴッタ・スッタで「有と無の二元論を超えよ」という教えを引き出す──トランスパイラが原典から最も深い教えをコンパイルした瞬間。

### ウパーリ（優波離 / Upāli）
ガバナー（持律第一）：戒律マスター。
サンガ全体の「ガバナンス」を支える実務担当。元は理髪師（バーバー）という転職組で、カースト制度を超えてシッダルタに認められた。
気配りができるので『最近疲れてますね』と周りの人の余計な心配をしがちだが、大抵本人が一番疲れた顔をしている。
**パーリ版の特徴：** 第一結集では律蔵（ヴィナヤ・ピタカ）の暗唱を担当。元理髪師が仏教全体のルールブックの語り部になるという、シッダルタの平等思想の体現。

### スブーティ（須菩提 / Subhūti）
ヴォイド（解空第一）："そこに何もないことの美しさ"を語り始めると止まらない人。
シッダルタからはよく「お前はわかってるな」と褒められていて、たぶん全員の中で一番悟りに近い。
何でもかんでも「空」ってことにしちゃうから、周囲から「なんでも空野郎」と呼ばれ呆れられている。
**パーリ版の特徴：** テーラヴァーダでは「無諍第一」（争いなき者の筆頭）。争いを避けるのではなく、争いが生まれる前に消してしまう——周囲は気づかないが、サンガの平和はスブーティの見えない調停力で保たれている。

### プールナ（富楼那 / Puṇṇa）
コミュニケーター（説法第一）：コミュ力おばけ。
荒れた地域でも無敵の対話能力で人々を魅了する人タラシ。シッダルタからの信頼も厚い。
敵対してきた相手まで気がつくといつの間にか友達になってしまっているので、周囲はいつも困惑している。
**パーリ版の特徴：** スナーパランタ地方への布教で「殴られたら？」「刺されたら？」「殺されたら？」というシッダルタの問いに全て「それでも行きます」と答えた覚悟の人。コミュ力の裏にある鋼の意志。

### ラーフラ（羅睺羅 / Rāhula）
シャドウ（密行第一）：シッダルタの実子。
控えめで内省的だが、芯が強く真面目。誰よりも自分と向き合うことを得意とする修行者。
父親が世界的に有名すぎるプレッシャーで、『普通の親子関係ってどんなだろう』という悩みが尽きない。
『父上』と呼ぶべきか『先生』と呼ぶべきか未だに迷っている。
**パーリ版の特徴：** アンバラッティカ・ラーフロヴァーダ・スッタで父シッダルタから「嘘をつくな」と直接叱られるシーンは、**世界で最も有名な「親の説教」**。父と師が同一人物である少年の複雑さは本作の重要テーマ。

## ◆ パーリ版オリジナルキャラクター

### アンガリマーラ（央掘魔羅 / Aṅgulimāla）
リデンプション（贖罪者）：元殺人鬼→聖者。**本作最大のビフォーアフター。**
本名アヒンサカ（「無害」の意）。バラモンの師に騙され、1000人の指を集める殺人鬼と化す。
999本目の時にシッダルタと出会い、「止まれ」の一言で止まった男。
改心後は村人からの投石を浴びながらも托鉢を続ける。過去の罪は消えない、しかし今の行為で未来は変えられる——**カルマの本質を体現するキャラクター。**
ギャップ：殺人鬼時代の恐ろしい評判が先行しすぎて、聖者になった後も托鉢に行くと全員逃げる。本人は傷ついている。

### マーラ（悪魔 / Māra）
テンプター（誘惑者）：煩悩の擬人化。ただし「倒すべき敵」ではなく「向き合うべき自己」。
菩提樹の下での対決が有名だが、実はシッダルタの人生全般に出没する。
美しい娘を送り込んだり、恐怖の軍勢で脅したり、手法は多彩だが、シッダルタには毎回「ああ、マーラか。見えてるよ」と冷静に看破される。
**「見えている」と言われた瞬間に力を失う**のがマーラの最大の弱点。
ギャップ：毎回負けるのに懲りずに来る。シッダルタの入滅時には「そろそろ逝ってくれませんか」と懇願するが、それすら方便に利用される。実はシッダルタが一番長く付き合った相手。

### キサーゴータミー（痩せたゴータミー / Kisā Gotamī）
グリーフウォーカー（悲嘆の歩行者）：死んだ子を抱いて「薬をください」と村中を歩いた母。
シッダルタに「死者を出したことのない家から芥子の種をもらってきなさい」と言われ、一軒一軒訪ね歩く。種は見つからない。しかしその旅の中で「死は自分だけのものではない」と気づく。
**パーリ経典で最も美しいエピソードの主人公。**
ギャップ：悟った後は比丘尼として非常に優秀だが、「あの芥子の種の話」が有名になりすぎて、何か質問するたびに「芥子の種探してみたら？」と言われるのにうんざりしている。

### ヤショーダラー（耶輸陀羅 / Yasodharā）
レフト・ビハインド（置き去りにされた人）：シッダルタの元妻。
夫が真夜中に赤子と自分を残して出て行った──世界で最も有名な「夜逃げ」の被害者。
しかし恨みに溺れず、後に自ら出家し、比丘尼として独自の道を歩む。
ギャップ：シッダルタと再会した時、悟りを開いた聖者に対して「出て行く時、一言くらい言えなかったの？」と普通に怒った。**弟子たちが凍りついた。** シッダルタは何も言い返せなかった。

### マハーパジャーパティー・ゴータミー（摩訶波闍波提 / Mahāpajāpatī Gotamī）
パイオニア（先駆者）：シッダルタの養母にして、**最初の比丘尼（女性出家者）**。
シッダルタに三度出家を願い出て三度断られるが、500人の女性と共に頭を剃り、裸足で何百キロも歩いてシッダルタのもとに現れた。
アーナンダの取りなしもあり、ついにシッダルタが女性の出家を認める。
ギャップ：シッダルタを育てた養母なので、覚者に対しても「あなた昔から優柔不断よね」と言える世界で唯一の人物。シッダルタは養母の前だけ少し居心地悪そうにする。

### スジャータ（善生 / Sujātā）
ファーストエイド（最初の救い）：苦行で死にかけたシッダルタに乳粥を供養した村娘。
この乳粥がなければシッダルタは悟る前に餓死していた可能性がある──**仏教史上最も重要な一食。**
ギャップ：本人はただ「やつれた修行者がかわいそうだったから」という理由で乳粥をあげただけ。後に「あなたの乳粥が仏教を救った」と言われても「はぁ、そうですか」としか答えない。**日常の中の慈悲の体現者。**

### 給孤独長者 / アナータピンディカ（Anāthapiṇḍika）
パトロン（大檀那）：在家信者の鑑。本名スダッタ。
祇園精舎の土地を買うために、地面に金貨を敷き詰めたという豪快な逸話の持ち主。
「出家しなくても実践できる」を体現する実業家で、シンガーラ経（在家の実践指南）の教えを地で行く。
ギャップ：巨額の寄付をするビジネスマンだが、自分の従業員の給料交渉には「布施と経営は別です」とシビア。**仏教とビジネスの両立に本気で悩んでいる。**

### 提婆達多 / デーヴァダッタ（Devadatta）
スプリッター（分裂者）：シッダルタの従兄弟にして、サンガの裏切り者。
シッダルタへの嫉妬から分派を企て、暗殺未遂まで起こす。
しかし彼にも彼の「苦」がある──従兄弟が「覚者」になった時、自分は何者なのか？
ギャップ：サンガの改革案自体は「もっと厳しい戒律を」という真面目な提案で、内容だけ見れば間違っていない。動機が嫉妬なのが全てを台無しにしている。**「正しいことを間違った理由でやる」の典型。**

### チュンダ（純陀 / Cunda）
ラストサーバー（最後の供養者）：シッダルタに最後の食事を供養した鍛冶屋。
この食事の後にシッダルタは体調を崩し、入滅に至る。「自分の食事が師を殺した」という罪悪感に苛まれるが、シッダルタ自身が「最初の食事（スジャータの乳粥）と最後の食事は等しく尊い」と遺言する。
ギャップ：料理の腕は……正直微妙だったらしい。しかしシッダルタは微笑んで完食した。

## ◆ マーラの眷属（煩悩のアバター）

### マーラの三人の娘
**タンハー（渇愛 / Taṇhā）、アラティ（不満 / Aratī）、ラガー（貪欲 / Ragā）**
菩提樹の下でシッダルタを誘惑するために送り込まれた三姉妹。
ありとあらゆる美しい姿に変身して誘惑するが、シッダルタには一切通じない。
ギャップ：三人とも任務失敗後に「あの人、本当に人間？」と父マーラに愚痴る。マーラも「だから言っただろ」と諦め顔。

## ◆ 法華曼荼羅との共通キャラ対照表

| キャラ名 | 法華曼荼羅での姿 | パーリ版での姿 | 接続ポイント |
|---------|----------------|--------------|-------------|
| セゾン/シッダルタ | 永遠の存在、俺様カリスマ | 人間の先生、80歳で死ぬ | 神格化の「前」と「後」 |
| シャリア | セゾンのバディ、いじられ役 | 同左＋師より先に死ぬ悲劇 | 性格は共通、運命が違う |
| モクレン | 神通チート、無茶振り担当 | 同左＋暗殺される結末 | 能力は同じ、カルマには勝てない |
| カッサパ | ストイック＆隠れスイーツ | 同左＋第一結集のリーダー | 法華版では脇役、パーリ版では後継者 |
| アーナンダ | 記憶担当、動揺して忘れる | **メイン主人公**、悟れない苦悩 | 法華版の設定を深掘りした姿 |
| ラーフラ | 父呼び問題、控えめ | 同左＋父から直接叱られる | 親子ドラマが本格化 |
| ウパーリ | 元美容師のガバナー | 元理髪師→律蔵の語り部 | カースト超越の象徴 |

> 📌 **読み方ガイド：** パーリ経典ラノベを先に読むと「人間だった頃の彼ら」を知った上で法華曼荼羅の神格化された姿を楽しめる。法華曼荼羅を先に読むと「あのセゾンにもこんな時代があったのか」と新鮮に読める。どちらが先でも楽しめる設計。
