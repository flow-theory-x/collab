# 安般念経（Ānāpānasati Sutta, MN 118）

## 概要
中部経典第118経。入出息の観察（ānāpānasati）を体系的に説く根本経典。呼吸瞑想の16段階を示し、それが四念処→七覚支→明知と解脱へと連鎖的に完成することを説く。パヴァーラナー（自恣）の満月の夜、サーヴァッティの東園で説かれた。

## 内容要約

### 背景設定
- 場所：サーヴァッティ（Sāvatthi）東園ミガーラマーター殿堂
- 長老比丘たちが新参比丘を10人、20人、30人、40人と教導していた
- ウポーサタの満月の夜、ブッダは僧団の静寂を見渡し大いに喜ぶ
- 僧団の中には阿羅漢・不還者・一来者・預流者が含まれる
- 「この修行に満足している。さらに努力せよ」と激励

### 呼吸瞑想の16段階
森・木の下・空き家に行き、結跏趺坐し、正念を前に置く。

#### 身に関する4段階（第1テトラッド）
1. 長い息を吸う時「長い息を吸う」と了知
2. 短い息を吸う時「短い息を吸う」と了知
3. 「全身を感受しつつ吸い・吐く」と修習
4. 「身行（呼吸）を鎮めつつ吸い・吐く」と修習

#### 受に関する4段階（第2テトラッド）
5. 「喜（pīti）を感受しつつ吸い・吐く」と修習
6. 「楽（sukha）を感受しつつ吸い・吐く」と修習
7. 「心行（受と想）を感受しつつ吸い・吐く」と修習
8. 「心行を鎮めつつ吸い・吐く」と修習

#### 心に関する4段階（第3テトラッド）
9. 「心を感受しつつ吸い・吐く」と修習
10. 「心を喜ばせつつ吸い・吐く」と修習
11. 「心を安定させつつ吸い・吐く」と修習
12. 「心を解放しつつ吸い・吐く」と修習

#### 法に関する4段階（第4テトラッド）
13. 「無常を観じつつ吸い・吐く」と修習
14. 「離貪を観じつつ吸い・吐く」と修習
15. 「滅を観じつつ吸い・吐く」と修習
16. 「捨離を観じつつ吸い・吐く」と修習

### 呼吸瞑想から四念処への連鎖
- 段階1-4 → 身随観（kāyānupassanā）：呼吸は「身の中の身」
- 段階5-8 → 受随観（vedanānupassanā）：呼吸への注意は「受の中の受」
- 段階9-12 → 心随観（cittānupassanā）
- 段階13-16 → 法随観（dhammānupassanā）

### 四念処から七覚支への連鎖
念処の修習 → 念覚支 → 択法覚支 → 精進覚支 → 喜覚支 → 軽安覚支 → 定覚支 → 捨覚支

各覚支は前の覚支から自然に生起する連鎖構造：
- 正念が安定すると → 択法が起こる
- 択法から → 精進が起こる
- 精進から → 喜が起こる
- 喜から → 身心の軽安が起こる
- 軽安から → 定（集中）が起こる
- 定から → 捨（平静）が起こる

### 七覚支から明知と解脱への完成
- 七覚支を遠離・離貪・滅に依拠して修習すると、明知（vijjā）と解脱（vimutti）が完成する

## 執筆で使えそうなポイント
- 呼吸瞑想の具体的手順——主人公が瞑想を習う場面の詳細な描写に
- 16段階の段階的深化——修行の「レベルアップ」を描く構造として最適
- 呼吸→四念処→七覚支→解脱の連鎖——一つの実践がすべてを完成させるという美しい構造
- 満月の夜に僧団が集う美しい場面設定——物語の重要シーンの背景に
- 「全身を感受しつつ呼吸する」——武道的な「全身の気づき」との接点
- ブッダが僧団を見渡して満足する冒頭の場面——師が弟子の成長を喜ぶ描写に
- 「喜→楽→心行の鎮まり」の流れ——瞑想体験のリアルな内的描写として
