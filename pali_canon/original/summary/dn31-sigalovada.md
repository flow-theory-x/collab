# 善生経（Sigālovāda Sutta, DN 31）

## 概要
長部経典第31経。「在家者の戒律」とも呼ばれ、在家仏教徒の倫理と社会的義務を最も体系的に説く経典。六方礼拝という古代インドの儀式を仏教的に再解釈し、人間関係の理想的なあり方を具体的に示す。テーラワーダ仏教圏で在家者向けの最重要経典。

## 内容要約

### 導入
- 場所：ラージャガハ、竹林精舎
- 居士の子シガーラが、亡父の遺言に従い濡れた衣・髪のまま六方（東南西北・下・上）を礼拝していた
- ブッダが托鉢途中にこれを見て、聖者の律における六方礼拝を教える

### （1）行為における四悪（四つの悪業）
- 殺生・偸盗・邪淫・妄語——これらを断つ

### （2）四つの悪の動機
- 欲（chanda）・瞋（dosa）・癡（moha）・恐怖（bhaya）に導かれて悪を為さない
- 月の満ち欠けの比喩：欲・瞋・癡・恐怖に従う者は月が欠けるように衰え、従わない者は月が満ちるように栄える

### （3）財産を散ずる六つの門
1. **酒類への耽溺** → 財産の損失、争いの増加、病気、悪評、恥ずべき行為、知力の低下
2. **夜間の徘徊** → 自身・妻子・財産が無防備に
3. **演劇見物への耽溺** → 「どこで踊りがある？歌がある？」と心が奪われる
4. **賭博への耽溺** → 勝者は怨みを生み、敗者は嘆き、法廷で信用されず、友に蔑まれ、縁談も来ない
5. **悪友との交際** → 博徒・放蕩者・酒飲み・詐欺師が友となる
6. **怠惰** → 「暑すぎる」「寒すぎる」「遅すぎる」「早すぎる」「腹が減りすぎ」「満腹すぎ」と言い訳

### 友についての教え

#### 敵に等しい四種の偽りの友
1. **収奪する者**：友の財を奪い、少なく与え多く求める
2. **口先だけの者**：過去のことで親しげにし、未来のことで親しげにし、空言で好意を得ようとし、いざという時は無力を訴える
3. **おべっか使い**：悪い行いを認め、良い行いを認めず、面前で褒め、背後で貶す
4. **破滅をもたらす者**：酒・夜遊び・演劇・賭博の仲間

#### 真の四種の温かい友
1. **助力者**：放逸な時に守り、財産を守り、危難の時の避難所となり、必要な時は倍額を提供
2. **苦楽を共にする者**：秘密を打ち明け、秘密を守り、不幸でも見捨てず、命さえ犠牲にする
3. **善き助言者**：悪から遠ざけ、善を勧め、未知のことを教え、天への道を示す
4. **同情者**：不幸を喜ばず、幸福を喜び、悪口を制し、善い評判を讃える

### 六方の仏教的再解釈
六方を六つの人間関係に割り当てる：

#### 東＝両親
- 子の五義務：養い、家業を継ぎ、家系の伝統を守り、相続に値する者となり、亡き親のために功徳を回向
- 親の五義務：悪から遠ざけ、善を勧め、職業を訓練し、適切な結婚を整え、適時に相続させる

#### 南＝師
- 弟子の五義務：起立して敬礼、師に仕え、学ぶ熱意、個人的奉仕、教えを敬聴
- 師の五義務：最善の訓練、教えの理解を確認、学芸を教え、友人に紹介、あらゆる方面の安全を図る

#### 西＝妻子
- 夫の五義務：礼を尽くし、蔑まず、貞節を守り、権限を委ね、装飾品を与える
- 妻の五義務：家事をよく行い、親族・使用人をもてなし、貞節を守り、財産を守り、勤勉に義務を果たす

#### 北＝友人・仲間
- 五義務：気前よく、丁寧な言葉で、助けとなり、公平で、誠実に
- 友の五義務：放逸を守り、財産を守り、危難の避難所となり、困難で見捨てず、家族を顧みる

#### 下方（地）＝使用人・雇用者
- 主人の五義務：能力に応じた仕事、食事と給与、病時の看護、美味の分かち合い、適時の休暇
- 使用人の五義務：主人より早く起き、遅く寝、与えられたもののみ取り、仕事をよくし、良い評判を守る

#### 上方（天）＝沙門・婆羅門
- 在家の五義務：慈愛ある行い・言葉・思い、家を開放し、物質的必要を満たす
- 沙門の六義務：悪から遠ざけ、善を勧め、慈愛の心で愛し、聞いていないことを教え、聞いたことを明確にし、天への道を示す

### 四つの和合の基礎（Saṅgahavatthu）
- **布施（dāna）・愛語（piyavacana）・利行（atthacariyā）・同事（samānattatā）**
- 「車軸の如く世界を回す」——これらなくして母も父も子の敬意を得られない

### シガーラの帰依
- 「素晴らしい、世尊よ！倒れたものを起こし、隠されたものを現し、道に迷った者に道を示し、暗闇に灯を掲げるようなものです」
- 仏法僧に帰依し在家信者となる

## 執筆で使えそうなポイント
- **在家者のための包括的な倫理体系**——異世界の社会制度を描く際の参考に
- 偽りの友と真の友の四分類——キャラクター造形に直接使える
- 六方の人間関係論——師弟・夫婦・親子・主従・友人の理想的関係を描く枠組み
- 賭博・酒・夜遊びの六害——堕落するキャラクターの描写に
- 怠惰の六つの言い訳——コミカルなキャラクター描写に
- 「車軸の如く世界を回す」四つの和合の基礎——為政者の心得として
- シガーラが父の遺言を形式的に守るのをブッダが内容的に再解釈する構造——形式と精神の対比
- アンバパーリーの場面と対照的に、こちらは「普通の若者」がブッダに出会う物語
