## 選定基準

「Theravada minimal」本書の目次に基づき、以下の基準で選定：

1. **実践可能性** — 現代人が日常で実践できる教え
2. **普遍性** — 宗教的信念を前提としない
3. **効果の明確性** — 実践による具体的変化が期待できる
